- **Etapa 3.**  Adicionando a coluna Escolaridade e atribuindo cada linha de forma aleat√≥ria.
- Para isso utilizei a fun√ß√£o .withColumn para criar uma nova coluna e para cada linha √© gerado um numero aleat√≥rio usando a fun√ß√£o rand() que gera valores aleat√≥rio entre 0 e 1, e como tenho 3 casos de escolaridade ent√£o 1/3 = 0.33, ou seja, tenho tres possiveis casos usando meu rand para escrever em cada linha. Para as possibilidades nos `when` tenho: se o n√∫mero for entre 0 a 0.33 a linha ser√° preenchida com o 'Fundamental', se for entre 0.33 e 0.66 ser√° preenchida com o 'Medio' e para outras possibilidades (`otherwise`) 'Superior'.
    ```python 
    #Criando a columa de Escolaridade com valores aleatorios
    df_nomes = df_nomes.withColumn(
        "Escolaridade",
        when(rand() < 0.33, "Fundamental")
        .when((rand() >= 0.33) & (rand() < 0.66), "Medio")
        .otherwise("Superior")
    )
    print("\n\n\n\n\n Data frame com coluna de Escolaridade")
    df_nomes.show(5)
    ```
- **Etapa 4.**  Adiconando coluna Pa√≠s
- Utilizando a mesma l√≥gica acima criei a coluna Pais que cont√©m 13 paises da Am√©rica do Sul, s√≥ que como tenho 13 valores, ent√£o 1/13 = 0.077 ent√£o continuei os possiveis 13 intevalos de casos de gera√ß√£o de numeros aleat√≥rio com rand() para cada linha.
    ```python
    #Criando a columa de Pais com valores aleatorios
    df_nomes = df_nomes.withColumn(
        "Pais",
        when(rand() < 0.07, "Brasil")
        .when((rand() >= 0.07) & (rand() < 0.15), "Argentina")
        .when((rand() >= 0.15) & (rand() < 0.23), "Bolivia")
        .when((rand() >= 0.23) & (rand() < 0.30), "Peru")
        .when((rand() >= 0.30) & (rand() < 0.38), "Uruguay")
        .when((rand() >= 0.38) & (rand() < 0.46), "Chile")
        .when((rand() >= 0.46) & (rand() < 0.53), "Venezuela")
        .when((rand() >= 0.53) & (rand() < 0.61), "Colombia")
        .when((rand() >= 0.61) & (rand() < 0.69), "Paraguai")
        .when((rand() >= 0.69) & (rand() < 0.76), "Equador")
        .when((rand() >= 0.76) & (rand() < 0.84), "Suriname")
        .when((rand() >= 0.84) & (rand() < 0.92), "Guiana")
        .otherwise("Guiana Francesa")
    )

    print("\n\n\n\n\n Data frame com coluna de Pais")
    df_nomes.show(5)
    ```

- **Etapa 5.**  Adicionando coluna AnoNascimento com valores aleat√≥rios entre 1945 e 2010
- Para gerar os valores aleat√≥rios de ano em cada linha no intervalo de ano de 1945 a 2010 com a seguinte l√≥gica:
  - A fun√ß√£o rand() gerar valores aleat√≥rios entre 0 e 1
  - Com o resultado entre 0 e 1, ser√° multiplicado pela soma de 2010 - 1945 + 1 correspondete ao valor de intervalo entre esses anos (no caso 66 anos). Ent√£o, se multiplicarmos qualquer valor de 0 a 1 por 66 o resultado vai dar um n√∫mero com parte inteira que somado com 1945 vai resultar em um ano corresponde ao intervalo desejado.
  - Ex: `0.55 * 66 = 36,3 => floor(36,3) = 36 => 36 + 1945 = 1981`
  - Por fim, converto cada linha da couna para o tipo inteiro.
    ```python
    df_nomes = df_nomes.withColumn(
        "AnoNascimento",
        (floor(rand() * (2010 - 1945 + 1)) + 1945).cast(IntegerType())
    )

    print("\n\n\n\n\n Data frame com coluna de AnoNascimento")
    ```

- **Etapa 10.**  Quantidade de pessoas de cada pa√≠s para cada gera√ß√£o com o uso de Spark SQL
    
- Essas foi uma das consultas que mais me dificultou no exerc√≠cio, pois tive que usar diversas opera√ß√µes de consultas e sub consulta.
- Primeiro tive que realizar uma sub consulta para gerar a coluna geracao, utilizei a fun√ß√£o condicional WHEN do sql para nomear cada linha para a gera√ß√£o pelo intervalo de ano de nascimento que corresponde que corresponde. Tive que nomer essas sub consulta para n√£o dar erros de ambiguidade.
    ```sql
        SELECT 
            Pais,
            CASE
                WHEN AnoNascimento BETWEEN 1944 AND 1964 THEN 'Baby Boomers'
                WHEN AnoNascimento BETWEEN 1965 AND 1979 THEN 'Gera√ß√£o X'
                WHEN AnoNascimento BETWEEN 1980 AND 1994 THEN 'Millennials'
                WHEN AnoNascimento BETWEEN 1995 AND 2015 THEN 'Gera√ß√£o Z'
            END AS Geracao
        FROM Pessoas
        AS tabelaComColunaGeracao
    ```

- Ap√≥s isso, realizei um consulta de pais, gera√ß√£o e quantidade de gera√ß√µes na sub consulta `tabelaComColunaGeracao`, agrupando pelas colunas pais e gera√ß√£o.
- Por fim, adicionei o resultado da consulta em um dataframe e depois ordenei usando a fun√ß√£o `.orderBy()` primeiramente pelo Pais, segundamente por gera√ß√£o e por fim por quantidade de cada gera√ß√£o.
    ```python
    df_result = spark.sql("""
        SELECT 
            Pais, 
            Geracao, 
            COUNT(*) as Quantidade
        FROM 
            (
                SELECT 
                    Pais,
                    CASE
                        WHEN AnoNascimento BETWEEN 1944 AND 1964 THEN 'Baby Boomers'
                        WHEN AnoNascimento BETWEEN 1965 AND 1979 THEN 'Gera√ß√£o X'
                        WHEN AnoNascimento BETWEEN 1980 AND 1994 THEN 'Millennials'
                        WHEN AnoNascimento BETWEEN 1995 AND 2015 THEN 'Gera√ß√£o Z'
                    END AS Geracao
                FROM Pessoas
            ) AS tabelaComColunaGeracao
        GROUP BY 
            Pais, Geracao
    """)

    # Ordenando o DataFrame pelos campos Pais, Geracao e Quantidade
    df_ordered_result = df_result.orderBy("Pais", "Geracao", "Quantidade")
    print("\n\n\n\n\n Quantidade de pessoas de cada pa√≠s para cada gera√ß√£o:")
    df_ordered_result.show(10)
    ```
